# thread-matter

- Purpose: Future embedded support for Thread and Matter-capable devices.
- Will contain: Platform glue, commissioning helpers, capability adapters, and device apps using 802.15.4 stacks.
- Responsibilities: Give Alice a native path into low-power mesh products without distorting MVP Wi-Fi firmware structure.
- Interfaces: Connects future C6, H2, nRF52, or STM32WB-class hardware to Alice contracts and provisioning flows.
- Status: planned.
