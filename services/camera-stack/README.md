# camera-stack

- Purpose: Future local-first camera and NVR subsystem.
- Will contain: Stream ingestion, storage, eventing, privacy policies, and camera device adapters.
- Responsibilities: Handle camera-specific media and retention workflows independently of core Home OS state management.
- Interfaces: Ingests RTSP/WebRTC-class streams and exposes events, thumbnails, and health signals to Home OS.
- Status: planned.
